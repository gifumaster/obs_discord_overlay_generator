# OBS Voice CSS Generator

Static tool for generating OBS custom CSS for Discord voice widget avatar replacements.

## GitHub Pages

This project is a plain static site. GitHub Pages can publish it directly from this repository.

### Option 1: Deploy With GitHub Actions

1. Create a GitHub repository.
2. Push all files in this folder to the repository root.
3. In GitHub, open `Settings -> Pages`.
4. Set `Source` to `GitHub Actions`.
5. The workflow in `.github/workflows/deploy-pages.yml` will publish the site automatically.

### Option 2: Deploy From Branch

1. Create a GitHub repository.
2. Push all files in this folder to the default branch.
3. In GitHub, open `Settings -> Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select the default branch and the `/ (root)` folder.

## Local Use

Open `index.html` in a browser.
