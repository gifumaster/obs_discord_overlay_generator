使い方

最短手順

  1. 共有設定でサイズや発話時の見た目を決めます。
  2. ユーザー設定でユーザーを追加します。
  3. 画像を読み込み、必要ならトリミングで表示範囲を調整します。
  4. Discord User ID と上下位置を必要に応じて調整します。
  5. 右側の `CSSをコピー` で生成結果をコピーします。
  6. OBS の Discord Streamkit Overlay のカスタムCSS欄へ貼り付けます。

JSON保存 / 読込を使うと、設定をバックアップしたり別環境へ持ち運べます。
